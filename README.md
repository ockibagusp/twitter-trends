## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Run Unit Tests with [Vitest](https://vitest.dev/)

#### Web UI
```sh
npm run test:ui
```

#### Terminal
```sh
npm run test:unit
```

or,

Example:

```bash
ls src/pages/__tests__/CopydanPaste.spec.js
```

```sh
npm run test:unit -t CopydanPaste
```
